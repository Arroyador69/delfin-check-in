# 📦 Assets de la App

## Archivos Necesarios

### Icono de la App
- **iOS**: `icon.png` (1024x1024px)
- **Android**: `icon.png` (1024x1024px) - se genera adaptiveIcon automáticamente

### Splash Screen
- **Splash**: `splash.png` (1284x2778px recomendado)

## Generar Assets

Puedes usar herramientas online como:
- https://www.appicon.co
- https://www.makeappicon.com

O crear manualmente con diseño en Figma/Sketch.

## Placeholder Temporal

Por ahora, puedes usar cualquier imagen 1024x1024px como placeholder.
EAS Build te avisará si faltan assets, pero puedes generar builds de prueba sin ellos.
